Elm Webpack Boilerplate
=======================

A boilerplate for Elm + Webpack


## Requirement

- Node v8 or later
- Yarn


## Installation

Run `yarn install` in root.


## Usage

### Build

```
yarn build
```

Outputed into `dist/`.

### Release

```
yarn release
```
Outputed into `dist/`.

### Development

```
yarn dev
```

Open `http://localhost:8080` in a browser.
